📱 نصب روی موبایل و تبلت
بدون دانلود — مستقیم از مرورگر نصب کنید

نت‌یار یک Progressive Web App (PWA) است. مثل اپ‌های App Store نصب می‌شود اما نیازی به App Store ندارد.

🔨 ساخت نسخه دسکتاپ (برای توسعه‌دهندگان)
برای ساخت فایل‌های نصب واقعی این مراحل را دنبال کنید
۱
نصب پیش‌نیازها: Node.js v16+ و npm لازم است
nodejs.org را دانلود کنید
۲
زیپ netyard-app را extract کنید، سپس: 
cd netyard-app
npm install
۳
اجرای محلی (برای تست): 
node server.js
# → http://localhost:1000
۴
ساخت اپلیکیشن Electron: 
# ویندوز (.exe):
npm run build:win

# مک (.dmg):
npm run build:mac

# لینوکس (.AppImage):
npm run build:linux

# همه با هم:
npm run build:all
فایل‌های نهایی در پوشه dist/ ذخیره می‌شوند
۵
ایمیل واقعی: در server.js اطلاعات Gmail App Password را وارد کنید
تنظیمات → امنیت → تایید دو مرحله‌ای → App Passwords
